This 48g Monako Glass puts Claude Code in front of your eyes so you can vibe code anywhere
============================================================

Monako Glass is a new wearable Linux-based heads-up display designed for hands-free AI coding, allowing users to interact with software through gesture controls and bone conduction microphone technology.

### Key Points
- The 48g device features an ARM Cortex A7 chipset, a 300mAh battery, and a waveguide display for mobile computing.
- Users can run AI coding agents like Claude Code and Codex, or utilize creative software including Blender and Unreal Engine.
- The Vision Engine translates hand and finger gestures into digital commands for navigating apps and scrolling through code.
- A bone conduction microphone on the nose bridge isolates user voice input from ambient noise for clearer AI interaction.
- Monako Glass is currently available for preorder at $19, with an expected retail price of $399 and a planned market launch in August.

### Why it Matters
This device represents a shift toward wearable, hands-free computing environments that integrate AI agents directly into the user's physical workspace. By combining open-source Linux architecture with gesture-based control, it offers a portable alternative to traditional desktop setups for developers and creative professionals.
