Breaking Into a Prison Tablet
============================================================

Tech enthusiast Hugh Jeffreys successfully bypassed the restrictive security measures on a discarded US prison tablet, revealing an Iview Optimus-C-8001 device running an outdated version of Windows 10.

### Key Points
- The tablet was encased in a sealed, opaque plastic shell designed to limit user access to prison-provided services via a locked-down kiosk mode.
- Physical access required using a hacksaw to breach the hermetically sealed housing, which contained foam and adhesive to prevent tampering.
- The device was identified as an Iview Optimus-C-8001 featuring an Intel Atom Z8350 processor and 2 GB of RAM.
- A faulty third-party charging board was the primary cause for the tablet being discarded as e-waste.
- Because the Windows 10 Home installation lacked drive encryption, the operating system was fully unlocked and restored to functional status.

### Why it Matters
This teardown highlights the hardware limitations and security configurations of specialized electronics used within the correctional system. It also demonstrates how discarded institutional technology can be repurposed by hobbyists once proprietary software restrictions are removed.
