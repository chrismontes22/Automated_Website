MicroSIP Portable 3.22.12 (lightweight SIP softphone) Released
============================================================

The latest version of MicroSIP Portable has been released, offering users a lightweight, open-source softphone for high-quality VoIP calls integrated directly into the PortableApps.com platform for mobile use.

### Key Points
- MicroSIP Portable enables high-quality audio and video communication using the open SIP protocol.
- The software supports major voice codecs, messaging, presence, and H.264 or H.263+ video standards.
- Security features include configurable TLS and SRTP encryption for both control and media traffic.
- The application is packaged in PortableApps.com format for seamless integration with portable drives and backup utilities.
- Users can download the free, open-source tool directly from the official MicroSIP Portable homepage.

### Why it Matters
This release provides a flexible, privacy-focused communication tool that users can carry on any portable device without requiring a local installation. By supporting open SIP standards, it offers a cost-effective alternative for international calling while maintaining compatibility with the broader PortableApps ecosystem.
