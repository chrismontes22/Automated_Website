Lingon and Lingon Pro 10
============================================================

The developer Peter Borg Apps has released Lingon 10 and Lingon Pro 10, updated macOS utilities that simplify scheduling apps, scripts, and commands through a user-friendly interface.

### Key Points
- Lingon 10 is a free, simplified version available on the Mac App Store, while Lingon Pro 10 is an advanced, one-time purchase standalone application.
- Both versions are optimized for macOS 14 Sonoma and later, featuring full support for Apple Silicon processors.
- Lingon Pro 10 includes advanced capabilities such as running tasks as root, background task monitoring, and Apple Intelligence integration for script creation.
- New features across both versions include natural language task creation, menu bar execution, and improved scheduling options.
- Lingon Pro 10 offers professional tools like smart folders, log filtering, and the ability to manage multiple tasks via tabs and windows.

### Why it Matters
These utilities provide a graphical interface for managing `launchd`, the system process responsible for background tasks on macOS, removing the need for users to interact directly with the Terminal. By streamlining automation, the software makes complex system scheduling accessible to general users while offering the power required by advanced administrators.
