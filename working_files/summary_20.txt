I found an Android TV launcher that just works better than Google's
============================================================

The new open-source Arc Launcher for Android TV offers a streamlined, ad-free interface inspired by Apple TV, providing users with a faster and more customizable alternative to Google TV.

### Key Points
- Arc Launcher is an open-source project designed to improve performance and simplicity on Android TV devices.
- The interface features a dock-based layout, custom app categories, and dynamic wallpaper options without intrusive ads or content recommendations.
- The launcher is currently unavailable on the Google Play Store and must be sideloaded via GitHub.
- Users often require third-party tools like Button Mapper to set custom launchers as the default due to Google's restrictive interface settings.

### Why it Matters
This development highlights growing user frustration with the cluttered, ad-heavy nature of the default Google TV interface. It underscores a significant demand for more user-centric, lightweight software options on streaming hardware that Google currently makes difficult to customize.
