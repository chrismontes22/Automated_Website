PicoZ80 – Drop-In Z80 Replacement
============================================================

The picoZ80 is a drop-in CPU replacement board for retro-computers that uses a Raspberry Pi RP2350 microcontroller to provide modern features like WiFi, Bluetooth, and SD-card storage.

### Key Points
- Designed as a DIP-40 socket replacement for Z80-based industrial and vintage computers.
- Utilizes a dual-core RP2350B processor running at up to 300MHz for cycle-accurate bus emulation.
- Includes an ESP32 co-processor for web-based management, file handling, and wireless connectivity.
- Features 8MB of PSRAM and 16MB of SPI Flash for extensive memory banking and ROM storage.
- Supports virtual device emulation and machine-specific "personas" via a JSON-based configuration system.
- Allows firmware updates and configuration changes through a browser-based interface without requiring recompilation.

### Why it Matters
This project enables owners of legacy hardware to modernize their systems with high-speed storage and network capabilities while maintaining original bus timing. By virtualizing peripherals and memory, it provides a flexible platform for hobbyists to preserve and extend the functionality of classic computing architectures.
