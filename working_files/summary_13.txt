Criminal IP and Securonix ThreatQ Collaborate to Enhance Threat Intelligence Operations
============================================================

Criminal IP has partnered with Securonix to integrate its threat intelligence data into the ThreatQ platform, enabling automated IP enrichment and faster security incident response for global organizations.

### Key Points
- Criminal IP’s threat intelligence is now integrated directly into the Securonix ThreatQ interface for real-time indicator validation.
- The integration provides automated data on maliciousness scores, VPN detection, open ports, and known vulnerabilities for IP addresses.
- Security teams can use ThreatQ’s orchestration engine to automate the evaluation of incoming threat indicators against Criminal IP’s database.
- The partnership aims to reduce manual analyst workloads by embedding exposure-based intelligence directly into existing investigation workflows.
- Users can perform on-demand lookups and visualize risk trends through unified dashboards without switching between different security tools.

### Why it Matters
This integration allows security operations centers to operationalize external threat data more efficiently by removing the need for manual context gathering. By embedding exposure-based intelligence into existing workflows, organizations can accelerate incident triage and improve the accuracy of their threat prioritization processes.
