Bluesky's Newest Product: an AI Tool That Gives You Custom Feeds
============================================================

**Bluesky has launched Attie, an experimental AI assistant that allows users to generate custom social media feeds using natural language prompts instead of traditional computer programming code.**

**Key points**
- Attie is built on the open-source AT Protocol, the same decentralized framework that powers the Bluesky social network.
- The tool enables users to create personalized content feeds by describing their preferences, such as specific topics or network interests, through conversational prompts.
- Beta signups for the new assistant are currently open to the public via the official Attie.AI website.
- Chief Innovation Officer Jay Graber stated that Attie operates as a separate application from the main Bluesky platform to facilitate experimentation with agentic social networking.

**Why it matters**
This development represents a shift toward making decentralized social media more accessible by removing the technical barrier of coding for feed customization. By empowering users to define their own digital experiences, the project aims to counter the influence of opaque, platform-controlled algorithms that currently dominate social media.
