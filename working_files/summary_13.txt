Here’s everything new for Apple Notes in iOS 27
============================================================

Apple’s iOS 27 update introduces significant enhancements to the Notes app, including AI-powered Siri integration, improved Markdown support, new organizational divider lines, and advanced Image Playground generative capabilities.

### Key Points
- Users can now insert horizontal divider lines into notes via the edit menu for better content organization.
- Siri AI integration allows users to seamlessly create new notes or append information to existing ones using voice commands.
- Markdown text pasted into Notes is now automatically converted to rich text, with a new option to copy content as Markdown.
- Image Playground features a new generative model capable of creating photorealistic images, which will include hidden SynthID watermarks.
- The Notes interface received visual refinements, including a refreshed app icon and updated iconography for shared folders and notes.

### Why it Matters
These updates streamline productivity by integrating generative AI directly into the note-taking workflow, reducing the need for manual formatting. By enhancing Markdown compatibility and organizational tools, Apple is positioning the Notes app as a more robust platform for professional writers and power users.
