OneDrive has a new generative AI photo tool — and it isn't called Copilot
============================================================

**One-sentence headline summary**
Microsoft has launched AI Restyle for OneDrive, a new feature that allows users to transform personal photos into various artistic styles using generative artificial intelligence technology.

**Key points**
* The AI Restyle feature is currently rolling out to OneDrive users on iOS, Android, and the web.
* Users can apply one-tap artistic styles, such as anime or cinematic posters, while retaining the original photo's core elements.
* Access to the tool requires a Microsoft 365 Premium subscription, which starts at $19.99 per month.
* The feature supports optional text prompts to further customize the image transformation process.
* Microsoft is releasing the tool gradually, meaning availability may vary based on user region and account status.
* Notably, the feature is branded as a standalone OneDrive tool rather than being marketed under the Microsoft Copilot umbrella.

**Why it matters**
This release signals a strategic shift in how Microsoft brands its generative AI tools by decoupling new features from the Copilot name. It also highlights the company's ongoing effort to integrate premium AI capabilities into its existing subscription ecosystem to drive value for Microsoft 365 users.
