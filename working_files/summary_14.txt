Warnings about removed or unsupported Play Store apps could head to Android
============================================================

Google is developing a new security feature for the Play Store that will alert Android users when installed applications have been removed from the platform and no longer receive updates.

### Key Points
- Tipster AssembleDebug discovered code strings in the latest Play Store version referencing apps removed from the marketplace.
- The system aims to notify users about apps that are no longer supported, potentially due to policy violations or developer removal.
- Alerts will specify the number of removed apps and encourage users to uninstall software that may pose security vulnerabilities.
- This initiative follows Google's recent implementation of warnings for apps that cause excessive battery drain on Android devices.

### Why it Matters
This feature enhances user security by identifying software that no longer receives critical patches or developer maintenance. By flagging these orphaned apps, Google helps users mitigate potential privacy risks and maintain a more secure mobile environment.
