Academic Research Skills for Claude Code
============================================================

The Academic Research Skills (ARS) suite provides an open-source, multi-agent AI pipeline designed to assist researchers with complex tasks like literature reviews, paper drafting, and peer review processes.

### Key Points
- The suite features a 13-agent research team and a 12-agent writing pipeline, supporting modes like Socratic dialogue, systematic review, and multi-perspective peer review.
- It integrates integrity gates, such as the 7-mode failure checklist and R&R Traceability Matrix, to prevent common AI issues like hallucinated citations and methodology fabrication.
- Users can install the suite via Claude Code CLI or VS Code using the command `/plugin marketplace add Imbad0202/academic-research-skills`.
- The system supports multiple citation formats, including APA 7.0, IEEE, and Chicago, and offers optional LaTeX compilation for professional-grade PDF output.
- Advanced features include Style Calibration to match a researcher's voice and a Devil's Advocate protocol to challenge arguments and improve logical consistency.

### Why it Matters
By automating the technical grunt work of research, this tool allows scholars to focus on high-level analysis and critical thinking while maintaining rigorous academic standards. It addresses structural AI limitations through explicit integrity checkpoints, making it a practical framework for researchers looking to leverage AI without sacrificing accuracy or authorship.
