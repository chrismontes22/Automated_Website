Thoth – open-source Local-first AI Assistant
============================================================

Thoth is a new open-source, local-first desktop AI assistant that provides users with personal data sovereignty, a structured knowledge graph, and autonomous agent capabilities without requiring cloud accounts.

### Key Points
- Runs locally via Ollama with 39 curated models or connects to external providers like OpenAI, Anthropic, and Google.
- Features a persistent knowledge graph that stores entities and relationships, exportable as an Obsidian-compatible wiki vault.
- Includes a Designer Studio for creating documents, decks, and media, plus 30 core tools for web browsing, shell access, and file management.
- Supports five messaging channels including Telegram, WhatsApp, and Slack, with approval-gated autonomous workflows.
- Offers one-click installers for Windows and macOS, with a one-line installer available for Linux systems.
- Operates without telemetry, account systems, or central servers, storing all API keys in the user's OS credential store.

### Why it Matters
Thoth offers a privacy-focused alternative to cloud-based AI assistants by keeping sensitive data and knowledge graphs entirely on the user's machine. This approach provides developers and power users with a highly customizable, autonomous toolset that maintains strict control over information flow and system access.
