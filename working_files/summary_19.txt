PowerToys just got 28% smaller and absorbed three separate apps I used to install
============================================================

Microsoft has released PowerToys version 0.100, featuring a significantly smaller installer and new integrated tools that replace several third-party applications for Windows 11 users to improve system efficiency.

### Key Points
- The PowerToys installer size has been reduced by 28%, dropping from 376MB to 272MB.
- The new Power Display tool allows users to control external monitor brightness and contrast directly, replacing apps like Twinkle Tray.
- Updated ZoomIt functionality now supports webcam overlays and video stitching, reducing the need for external recording software like OBS.
- The Grab and Move feature enables window resizing and movement from anywhere on a window, eliminating the need for third-party utilities like AltDrag.
- The Command Palette now includes an Extension Gallery, simplifying the discovery and installation of productivity-enhancing plugins.

### Why it Matters
These updates consolidate multiple niche utilities into a single, official Microsoft-supported suite, reducing software bloat and system complexity for Windows users. By integrating these features, Microsoft is effectively standardizing advanced desktop management tools that were previously only available through fragmented third-party ecosystems.
