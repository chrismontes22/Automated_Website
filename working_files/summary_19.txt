Lat.md: Agent Lattice: a knowledge graph for your codebase, written in Markdown
============================================================

**The new Lat tool introduces a markdown-based knowledge graph system that helps developers and AI agents maintain codebase documentation, design decisions, and test specifications through interconnected files.**

**Key points**
- Lat organizes project documentation into a `lat.md/` directory using wiki-style links to connect markdown files and specific source code symbols.
- The CLI tool includes `lat check` to enforce referential consistency between documentation and code, preventing information drift as projects evolve.
- Developers can annotate source code with `@lat` comments to create bidirectional links between implementation logic and architectural concepts.
- The system supports semantic search via OpenAI or Vercel AI Gateway, allowing agents to retrieve context faster than traditional grep-based methods.
- Test specifications can be enforced by requiring code mentions, ensuring that documentation and test coverage remain synchronized.

**Why it matters: **
This tool addresses the common problem of "documentation rot" where codebase context is lost or becomes outdated as projects scale. By integrating documentation directly into the development workflow, it enables AI agents to operate with higher accuracy and helps human teams maintain a clear record of architectural decisions.
