Pencil Launches Code on Canvas for AI Design Tool
============================================================

Pencil has launched Code on Canvas, an update to its AI design platform that allows Claude and Codex agents to build custom interactive tools directly within the workspace.

### Key Points
- The update enables AI agents to generate custom instruments, such as data components and generative art brushes, directly on the user's design canvas.
- Pencil utilizes an MCP-driven architecture where .pen design files reside within the codebase, ensuring design and code remain synchronized without manual handoffs.
- Users retain full manual control over the canvas, allowing for a hybrid workflow between human designers and autonomous AI agents.
- Pencil reached a milestone of 100,000 users in February 2026 following the release of its SWARM mode for parallel agent collaboration.

### Why it Matters
This integration shifts the role of AI from a passive design generator to an active developer of the design environment itself. By allowing agents to build tools on the fly, Pencil streamlines complex workflows and bridges the gap between static design files and functional application code.
