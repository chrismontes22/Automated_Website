Kimi K2.7 Code is generally available in GitHub Copilot
============================================================

GitHub has integrated the open-weight Kimi K2.7 Code model into its Copilot platform, providing developers with a new, cost-effective alternative for coding workflows across multiple integrated development environments.

### Key Points
- Kimi K2.7 Code is the first open-weight model available as a selectable option within the GitHub Copilot model picker.
- The model is hosted on Microsoft Azure and billed according to provider list pricing based on individual usage.
- Initial rollout includes Copilot Pro, Pro+, and Max plans, with support for Business and Enterprise tiers arriving in coming weeks.
- Users can access the model via Visual Studio Code, Visual Studio, JetBrains, Xcode, Eclipse, and the Copilot CLI.
- Business and Enterprise administrators must manually enable the Kimi K2.7 Code policy before the model becomes available to their organizations.

### Why it Matters
This integration marks a significant shift by offering developers greater flexibility and lower-cost options through open-weight model support. It allows organizations to balance performance needs with budget constraints while maintaining control over security and data governance policies.
