WhatsApp now testing its encrypted cloud storage backup solution for iOS
============================================================

WhatsApp is testing a new encrypted cloud storage feature for iOS and Android that allows users to back up chat history independently of iCloud or Google Drive services.

### Key Points
- The feature, discovered in iOS beta version 26.28.10.16, provides users with 2GB of free, end-to-end encrypted cloud storage.
- WhatsApp is reportedly considering a paid tier offering 50GB of storage for $0.99 per month.
- Backups secured via the new system can be protected using a passkey, a custom password, or a 64-digit encryption key.
- This development aims to provide an alternative to existing platform-specific backup solutions like Google Drive and iCloud.
- A public release date for the feature has not yet been announced by the Meta-owned company.

### Why it Matters
This feature could significantly simplify the process of migrating chat history between iOS and Android devices, which is currently hindered by fragmented backup systems. By offering a platform-agnostic storage solution, WhatsApp may reduce user reliance on third-party cloud providers while strengthening data privacy through mandatory end-to-end encryption.
