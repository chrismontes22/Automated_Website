Perplexity Launches Personal Computer for Mac, Turning a Mac mini Into an Always-On AI Agent
============================================================

Perplexity has officially launched Personal Computer, a new Mac-integrated tool for Max subscribers that automates local workflows and manages files using advanced AI agents and native application access.

### Key Points
- Personal Computer is now available to Perplexity Max subscribers, priced at $200 per month, with priority given to existing waitlist members.
- The software requires macOS 14 Sonoma or later and integrates directly with local file systems and native Mac applications.
- Users activate the tool by pressing both Command keys, allowing the AI to execute tasks via text or voice commands.
- The system utilizes teams of agents across more than 20 frontier models to perform complex workflows like file organization and data comparison.
- Security features include a sandboxed environment, an auditable action log, and a manual kill switch for user oversight.
- Perplexity recommends running the software on a Mac mini to support 24/7 operations and persistent local access.

### Why it Matters
This release marks a significant shift toward autonomous local computing by allowing AI to interact directly with a user's private files and desktop applications. By enabling persistent, cross-app automation, Perplexity is positioning its high-tier subscription as a comprehensive digital workforce for power users and businesses.
