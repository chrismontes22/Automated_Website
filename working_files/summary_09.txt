Files
============================================================

Windows 11 File Explorer provides a modernized, tabbed interface for managing local files and deep cloud integration with Microsoft OneDrive to streamline storage, synchronization, and secure data sharing.

### Key Points
- File Explorer features a tabbed interface, a customizable Home page, and a new Details pane for viewing file metadata.
- Native support now includes creating and extracting ZIP, TAR, and 7-ZIP archive formats without third-party software.
- OneDrive integration enables "Files On-Demand," allowing users to access cloud files without occupying local disk space.
- Users can manage folder backups for Desktop, Documents, and Pictures, or secure sensitive data using the Personal Vault.
- Advanced customization options are available via the Folder Options dialog or the Windows Settings app for power users.
- OneDrive can be unlinked or uninstalled through system settings if users prefer to manage files exclusively on local storage.

### Why it Matters
Understanding File Explorer’s integration with OneDrive is essential for managing storage efficiency and ensuring data remains accessible across multiple devices. Mastering these tools allows users to optimize their workflow, secure sensitive information, and maintain control over how their files are synced and stored.
