Claris CEO Ryan McCann on FileMaker in the Age of Agentic Coding
============================================================

Claris is integrating AI-assisted coding tools into the FileMaker platform, allowing developers to generate secure, production-ready applications by leveraging external agents for interface design and script creation.

### Key Points
- Claris is positioning FileMaker as a primary deployment target for agentic coding tools like Claude Code, Cursor, and Codex.
- The integration allows AI agents to generate web-native UI and native FileMaker scripts that inherit existing security and permission structures.
- FileMaker 2026 will introduce new infrastructure features, including native disaster recovery, remote backup, and standby server capabilities.
- Developer previews for the new agentic coding functionality are scheduled for release later this summer.
- The strategy aims to reduce development timelines from weeks to hours by automating complex layout and logic tasks.

### Why it Matters
As AI makes code generation trivial, the ability to manage, secure, and deploy software becomes the primary challenge for businesses. By bridging the gap between rapid AI development and enterprise-grade infrastructure, Claris ensures that developers can maintain professional standards while significantly increasing their productivity.
