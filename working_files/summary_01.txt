Proton Drive client is (finally) coming to Linux
============================================================

Proton has officially confirmed that it is developing a native Proton Drive client for Linux desktops, utilizing a new unified software development kit to streamline cross-platform feature integration.

### Key Points
- Proton is building the Linux client from the ground up using a new shared SDK that powers its Windows, macOS, and mobile applications.
- The transition to a unified codebase has improved performance, with reported upload speeds up to 3x faster and encryption speeds up to 4x faster.
- While the company confirmed development is underway, no specific release date or beta testing timeline has been announced for the Linux version.
- Proton Drive currently offers 5 GB of free encrypted cloud storage, with paid subscription tiers starting at £3.99 per month for 200 GB.
- The new SDK is available on GitHub under an MIT license, supporting TypeScript, C#, Kotlin, and Swift for future development.

### Why it Matters
The development of an official Linux client signals Proton's commitment to supporting open-source desktop environments, which have historically lacked native integration for the service. This unified SDK approach also ensures that future feature updates and performance improvements will be deployed more consistently across all major operating systems.
