This Week in Security: Microsoft on Microsoft, Register Your Domains, Linux on ARM, and FreeBSD Joins the File Cache Club
============================================================

Microsoft is addressing a surge in supply chain attacks as the Miasma worm compromises open-source repositories, prompting major security updates across GitHub, NPM, and Windows platforms.

### Key Points
- GitHub automatically disabled 73 Microsoft-related repositories, including Azure packages, following infections by the Miasma supply chain worm.
- NPM 12 will block automatic execution of install-phase scripts and remote URL dependencies to mitigate the spread of malicious packages.
- Microsoft’s latest Patch Tuesday addressed over 200 security vulnerabilities, including exploits for Windows Defender and BitLocker.
- The researcher known as MSNightmare released new Windows exploits, including the RoguePlanet vulnerability, following a public dispute with Microsoft.
- A Python linter, Ruff, successfully blocked a malicious code injection attempt by rejecting commits that violated project style guidelines.
- New vulnerabilities were identified in OpenSSL, the Linux kernel on Arm processors, and the FreeBSD kernel’s page-cache system.

### Why it Matters
These incidents highlight the growing fragility of the software supply chain as automated malware and credential-harvesting worms increasingly target developer environments. Organizations must prioritize robust dependency management and automated security patching to defend against these sophisticated, high-volume attacks.
