Android 17 Beta 3 finally brings the desktop multitasking we’ve been waiting for
============================================================

**Google has released Android 17 Beta 3 for Pixel devices, introducing a functional multitasking "Bubbles" system, separated Wi-Fi and mobile data toggles, and improved screen recording tools.**

**Key points**
- Android 17 Beta 3 has reached platform stability, allowing developers to finalize apps for the Google Play Store.
- The new "Bubbles" feature enables users to run most applications in floating windows for enhanced multitasking.
- Quick Settings now feature separate, dedicated tiles for Wi-Fi and mobile data instead of a combined internet menu.
- Screen recording includes a new floating toolbar with integrated options to edit, share, or delete captured videos.
- Accessibility updates allow users to route specific system sounds, such as alarms or ringtones, directly to hearing aids.
- New privacy controls include session-based location sharing and improved password visibility settings for physical keyboards.

**Why it matters: **
This update marks a significant shift toward desktop-like productivity on mobile devices by streamlining core navigation and multitasking workflows. Reaching platform stability signals that the operating system is nearing its final release, providing developers and users with a reliable foundation for upcoming features.
