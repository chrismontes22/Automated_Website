IR Device Control That Lives Off The Cloud
============================================================

The new IR2MQTT project provides a cloudless, open-source solution for integrating legacy infrared appliances into Home Assistant setups using ESP32 or ESP8266 hardware and automated code databases.

### Key Points
- IR2MQTT allows users to control older televisions and air conditioners without relying on proprietary, cloud-based smart home platforms.
- The system utilizes ESP32 or ESP8266 microcontrollers running ESPHome firmware to transmit infrared signals to household devices.
- A built-in web interface eliminates the need for manual IR code capture or complex YAML configuration files.
- The software integrates with extensive public databases, enabling quick setup for a wide range of common consumer electronics.

### Why it Matters
This tool offers a privacy-focused alternative for smart home enthusiasts who want to automate legacy hardware without external server dependencies. By simplifying the integration process, it lowers the technical barrier for users looking to consolidate disparate devices into a unified, local control ecosystem.
