Stellar Cyber improves threat detection and data onboarding in new updates
============================================================

Stellar Cyber has released versions 6.5 and 6.6 of its security operations platform, introducing governed AI workflows, enhanced threat detection, and self-service data onboarding for enterprise security teams.

### Key Points
- The updates introduce Model Context Protocol support, allowing secure integration of AI assistants directly into existing security operations workflows.
- Auto Triage visibility is improved with new verdict columns and response action panels, enabling analysts to filter and act on threats more efficiently.
- Detection capabilities are expanded across identity, cloud, and network environments, including new AWS Config monitoring and Microsoft Graph Security API integrations.
- Parser Studio provides a self-service workspace for teams to create and test custom data parsers, accelerating the onboarding of new security data sources.
- Platform operations are bolstered by a new Dashboard Hub, improved system health monitoring, and enhanced automation readiness for MSSPs and enterprise users.

### Why it Matters
These updates aim to reduce alert fatigue by integrating AI-driven context directly into the analyst workflow rather than relying on disconnected external tools. By streamlining data onboarding and detection fidelity, the platform helps security teams respond to threats faster while maintaining human oversight.
