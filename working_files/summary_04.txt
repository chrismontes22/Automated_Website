Accessibility settings are looking at a few refinements in this One UI 9 leak
============================================================

**Samsung is reportedly developing new accessibility features for its upcoming One UI 9 software, including enhanced text highlighting tools and improved mouse and keyboard navigation for Galaxy devices.**

**Key points**
- Leaked code suggests a "Text Spotlight" feature will allow users to expand and highlight text in a separate window with customizable size and color settings.
- New mouse and keyboard accessibility updates include cursor speed fine-tuning and physical keyboard shortcuts to toggle settings without navigating menus.
- Interaction menu changes will move single-tap and swipe options to the top level for easier access.
- One UI 9 will be based on Android 17 and is currently in early internal testing stages at Samsung.
- The software update is expected to debut on Samsung's next generation of foldable smartphones.

**Why it matters: **
These accessibility improvements demonstrate Samsung's ongoing commitment to inclusive design by simplifying navigation for users with vision or motor impairments. Refining these core interface elements ensures that the latest Android 17-based software remains functional and approachable for a broader range of Galaxy device owners.
