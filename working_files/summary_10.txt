Product showcase: Proton Authenticator is an end-to-end encrypted, open source 2FA app
============================================================

Proton has launched a free, open-source two-factor authentication app that provides end-to-end encrypted code generation and cross-platform synchronization for users across Windows, macOS, Linux, iOS, and Android devices.

### Key Points
- The app generates time-based one-time passwords (TOTP) that refresh every 30 seconds for secure account verification.
- Users can import existing tokens from services like Google Authenticator, Aegis, and Bitwarden to simplify the migration process.
- Data synchronization is protected by end-to-end encryption, ensuring that only the user can access their stored authentication tokens.
- The software functions offline, allowing users to generate codes without an active internet connection.
- Security features include optional biometric authentication or PIN protection to prevent unauthorized access to the app.
- The open-source codebase allows for public security audits and provides transparency regarding how the application handles sensitive user data.

### Why it Matters
This release provides a privacy-focused alternative to mainstream authentication tools by emphasizing user control and transparent, open-source security standards. By offering seamless cross-platform syncing and data portability, Proton aims to reduce the friction associated with managing multi-factor security across various digital services.
