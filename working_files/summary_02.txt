Croc: Securely transfer files and folders between two computers
============================================================

Croc is a secure, cross-platform command-line tool that enables users to transfer files and folders between computers using end-to-end encryption and password-authenticated key agreement without requiring port forwarding.

### Key Points
- Facilitates secure data transfers between Windows, Linux, and macOS systems using a relay-based architecture.
- Utilizes Password-Authenticated Key Agreement (PAKE) to ensure end-to-end encryption for all transmitted files and folders.
- Supports advanced features including interrupted transfer resumption, proxy integration via Tor, and IPv6-first connectivity.
- Offers flexible installation options across major package managers like Homebrew, Scoop, Chocolatey, Winget, Nix, and Docker.
- Allows users to host custom relays to maintain full control over their data transfer infrastructure and security parameters.

### Why it Matters
Croc simplifies the often complex process of moving data between disparate systems by removing the need for local server configuration or firewall adjustments. Its focus on security and ease of use provides a robust alternative for developers and power users who require a reliable, encrypted method for peer-to-peer file sharing.
