Microsoft CEO Satya Nadella says AI tokenmaxxing is costly: "I'm a tokenmaxxer too, it's addictive."
============================================================

Microsoft CEO Satya Nadella confirmed the company utilizes extensive AI-driven "tokenmaxxing" for productivity while mandating a transition from Claude Code to GitHub Copilot CLI by June 30.

### Key Points
- Microsoft is phasing out all employee Claude Code licenses effective June 30 to prioritize the internal GitHub Copilot CLI tool.
- CEO Satya Nadella stated that up to 30% of the company's software code is currently generated using artificial intelligence.
- The company is restricting the use of Anthropic’s Claude Fable 5 model due to specific data retention and protection requirements.
- Nadella advised employees to avoid using expensive frontier models for simple tasks, recommending Microsoft Copilot’s Auto Mode for better economic efficiency.

### Why it Matters
This shift highlights Microsoft's strategic effort to manage the rising operational costs associated with widespread generative AI adoption within its workforce. By steering employees toward internal tools, the company aims to balance high-level productivity gains with more sustainable infrastructure spending.
