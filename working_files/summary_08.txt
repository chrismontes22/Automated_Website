I'm impressed that Microsoft keeps on improving PowerToys with smart additions — but I wonder why the newest monitor trick isn't in Windows 11 by default
============================================================

Microsoft has released PowerToys version 0.99, introducing the Power Display and Grab and Move utilities to enhance monitor control and window management for Windows 10 and 11 users.

### Key Points
- Power Display allows users to adjust monitor brightness, contrast, and color temperature directly from the Windows system tray.
- The new tool utilizes DDC/CI communication to adjust hardware settings rather than applying software-based filters.
- Power Display integrates with the Light Switch utility to automate monitor profiles based on light and dark mode schedules.
- The Grab and Move utility enables users to resize or drag windows by holding the Alt key while clicking anywhere on the window.
- Both tools are currently in preview and require compatible hardware, specifically monitors that support DDC/CI for full functionality.

### Why it Matters
These additions highlight Microsoft's ongoing effort to refine the Windows 11 user experience through modular, open-source software. By testing these features in PowerToys, the company can gauge user demand for potentially integrating these highly requested productivity tools directly into the core operating system.
