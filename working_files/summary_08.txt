Off-Grid OCR Server Powered by iPhone
============================================================

Developer Hemant successfully repurposed an iPhone 8 as a solar-powered optical character recognition server, demonstrating that older mobile hardware can effectively handle intensive machine learning tasks off-grid.

### Key Points
- The project utilizes Apple’s Vision framework to perform character recognition on an iPhone 8 instead of traditional rack-mounted server hardware.
- A solar panel and external power bank provide a sustainable, off-grid energy source for the device.
- The primary technical challenge involved preventing the iOS operating system from putting the server application into sleep mode.
- Hemant estimates the setup saves approximately $10 in monthly electricity costs, offering a full return on investment within two to three years.
- This build prevents electronic waste by extending the functional lifespan of a smartphone that would otherwise be discarded.

### Why it Matters
This project highlights the potential for repurposing aging consumer electronics as low-power, cost-effective alternatives to traditional server hardware. By leveraging native machine learning frameworks, users can achieve significant energy savings and reduce electronic waste while maintaining functional computing infrastructure.
