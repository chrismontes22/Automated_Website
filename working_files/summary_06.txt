This free launcher made my smart TV feel like it should have all along
============================================================

Users can replace the cluttered Google TV home screen with Arc Launcher, a free, open-source interface that removes advertisements and provides extensive customization options for a streamlined streaming experience.

### Key Points
- Arc Launcher is an open-source, ad-free interface based on FLauncher that removes distracting trailers and suggested content.
- The software allows users to organize apps into custom categories, adjust status bar information, and set dynamic wallpapers.
- Advanced features include automatic brightness adjustment, a built-in screensaver with OLED burn-in protection, and detailed data usage monitoring.
- Users can bypass the default Google TV home screen by remapping the remote's Home button using the free Button Mapper application.
- The launcher supports sideloaded apps and provides granular control over app management, including hiding or uninstalling software directly from the home screen.

### Why it Matters
This alternative launcher addresses common user frustrations regarding intrusive advertisements and rigid navigation on the Google TV platform. By providing a lightweight, customizable interface, it empowers users to optimize their hardware for better performance and a more personalized viewing environment.
