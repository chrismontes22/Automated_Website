Google is readying a major autofill upgrade for Chrome
============================================================

Google is testing a new "Find and Fill with Gemini" feature in Chrome Canary that uses AI to pull personal information from Gmail and Photos for form autofill.

### Key Points
- The feature is powered by Google’s Personal Intelligence, which connects data across various Google apps.
- Users can trigger the tool by typing "@@" in a text field or by using a custom keyboard shortcut.
- The upgrade allows Chrome to retrieve information from external apps without requiring users to leave their current webpage.
- Privacy settings allow users to disable the feature within the "Enhanced Autofill" menu in Chrome settings.
- Google notes that page titles and URLs where the feature is activated may be reviewed by humans to improve the service.
- The functionality is currently in limited testing and is not yet available to all Chrome Canary users.

### Why it Matters
This integration represents a significant shift toward using generative AI to automate routine browser tasks by centralizing personal data access. While it offers increased convenience, the potential for human review of browsing data highlights ongoing privacy trade-offs in AI-driven browser features.
