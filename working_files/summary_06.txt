Windows 10 Still Being Used, Often Unpatched and Insecure
============================================================

Windows 10 remains installed on 16.9% of monitored devices as migration efforts stall, leaving millions of systems vulnerable to security threats due to a lack of necessary hardware upgrades.

### Key Points
- Lansweeper data shows 16.9% of Windows devices still run Windows 10, with migration rates slowing significantly.
- Small and medium-sized businesses report a 21.4% usage rate, while healthcare and pharmaceutical sectors reach 23%.
- Windows 10 devices average 1,903 active CVEs, nearly triple the 652 vulnerabilities found on Windows 11 systems.
- Only 14% of remaining Windows 10 assets have applied Extended Security Update patches to mitigate risks.
- Many industrial, medical, and retail systems cannot upgrade because they lack Windows 11-certified software versions.

### Why it Matters
The persistent use of legacy operating systems creates significant cybersecurity risks, as attackers can use Windows 11 patches to identify vulnerabilities in unsupported Windows 10 software. This trend highlights a critical infrastructure gap where cost constraints and specialized hardware requirements prevent organizations from completing necessary security migrations.
