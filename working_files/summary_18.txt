I found a hidden way to limit RAM on Windows 11, and the results were way more surprising than I expected
============================================================

Windows 11 users can restrict total system memory for testing and debugging purposes by adjusting the Maximum memory setting within the legacy System Configuration tool, also known as msconfig.

### Key Points
- Access the memory limit setting by searching for "msconfig" and navigating to the Boot tab's Advanced options.
- Users must input the desired memory limit in megabytes, such as 4096 for 4GB or 8192 for 8GB.
- A system restart is required to apply the memory restriction or to restore full RAM access.
- Modern systems with UEFI firmware and Secure Boot enabled may require temporarily disabling security features to modify these settings.
- This configuration is strictly for testing or troubleshooting and is not recommended for improving everyday system performance.

### Why it Matters
Artificially limiting RAM is a specialized diagnostic technique used by developers and IT professionals to simulate low-memory environments without hardware changes. Using this setting for general use is counterproductive, as it significantly degrades system performance and restricts the operating system's ability to manage resources efficiently.
