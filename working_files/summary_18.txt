DarkSword leak puts millions of iPhone users at risk
============================================================

**A sophisticated iPhone hacking tool known as DarkSword has been leaked online, significantly lowering the barrier for attackers to compromise devices running outdated versions of Apple's iOS software.**

**Key points**
- The DarkSword tool, which uses basic web technologies like HTML and JavaScript, is now publicly accessible to users with minimal technical expertise.
- Vulnerable devices include iPhones and iPads running iOS 18 or earlier, potentially exposing hundreds of millions of users to data theft.
- Successful exploits allow attackers to extract sensitive information, including contacts, messages, call history, and saved passwords from the iOS keychain.
- Apple released security updates on March 11, 2026, to patch the vulnerability, including specific protections for older devices running iOS 13 and 14.
- Users are advised to immediately install the latest software updates and consider enabling Lockdown Mode to mitigate potential risks.

**Why it matters: **
The public release of this hacking tool transforms a specialized threat into a widespread risk for any user who has not updated their device software. This incident highlights the critical importance of timely OS updates in defending against automated, mass-scale cyberattacks.
