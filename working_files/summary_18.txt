Claude Design System Prompt
============================================================

The open-source Claude Design system prompt provides a structured framework for LLMs to generate professional, accessibility-focused web designs while actively avoiding common AI-generated aesthetic tropes and generic templates.

### Key Points
- The project includes a 20-chapter system prompt and 14 procedural skills for production, extraction, and design review.
- It is MIT-licensed and compatible with major LLMs, including Claude, GPT, and Gemini, as well as local models.
- The framework enforces strict design principles, including WCAG accessibility compliance, semantic HTML, and intentional visual hierarchy.
- An integrated "AI-slop check" identifies and rejects common generative design patterns like excessive gradients, emojis, and rounded-corner cards.
- Skills are categorized into production, system extraction, and audit workflows, allowing users to chain tasks for complex design projects.

### Why it Matters
This tool offers a standardized approach to professionalizing AI-generated web design by replacing generic, repetitive outputs with disciplined, system-oriented code. By providing a modular framework for quality control, it enables developers and designers to use LLMs as reliable collaborators for building accessible, high-quality digital interfaces.
